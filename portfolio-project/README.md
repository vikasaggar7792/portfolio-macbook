# Vikas Aggarwal — Portfolio (VikasOS)

## Files
- `index.html` — page structure
- `style.css` — all styling
- `script.js` — all interactivity (windows, dock, terminal, music player, etc.)
- `assets/vikas.jpg` — your photo, used in Photos / LinkedIn / X / Notes / Lock screen

## How to run in VS Code
1. Open this folder in VS Code (`File → Open Folder`).
2. Install the **Live Server** extension (if you don't have it) — search "Live Server" by Ritwick Dey in the Extensions tab.
3. Right-click `index.html` → **Open with Live Server**.
4. It'll open in your browser at `http://127.0.0.1:5500` (or similar) — fully working, exactly like the version you tested.

You can also just double-click `index.html` to open it directly in a browser without VS Code — it works the same way, no server needed.

## Notes
- Everything is plain HTML/CSS/JS — no build step, no npm install needed.
- Songs play as a simulated player (progress bar, play/pause) since real copyrighted audio can't be hosted — swap in your own licensed audio files if you want real playback.
- Update your real bio, hobbies, or resume details directly inside `script.js` (search for the relevant text) whenever you want.
